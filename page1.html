CSS is great!

CSS stand for Cascading Style Sheets Presentation of content should be separate from the content itself.

CSS provide for a way to quickly change the style of elements in a single document or multiple document. The style changes can 'cascade' down the hierarchy.

The concept of cascade essentially means that we can provide specificity to the rules causing some rules to override others.
